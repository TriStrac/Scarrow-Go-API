  clearSerial2();
  Serial.print("[R2 "); Serial.print(label); Serial.print("] ");
  UART2.write(cmd, len);

  unsigned long start = millis();

    if (UART2.available()) resp[idx++] = UART2.read();

## Notes

- LD2420 outputs text (`Range XX`) after the binary config sequence (open + mode + close), not the `setRunMode` alone
- The three raw `UART2.write()` calls before `sendCmd2()` are what activate text output mode
- `sendCmd2()` reads the ACK/NAK response byte(s) optionally — not required for operation
- ESP32 UART2 supports custom pins via `HardwareSerial.begin(rx, tx)` on any available GPIO pair
- The OT1 pin change from GPIO 4 to GPIO 13 is the **critical fix** matching the user's actual wiring
